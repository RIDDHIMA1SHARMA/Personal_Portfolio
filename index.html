Detection of Diabetic Retinopathy Using Deep Learning

Diabetic Retinopathy is a leading cause of vision loss. Our project uses convolutional neural networks (CNNs) to classify retinal images into severity levels.

Key Features

  • Dataset: APTOS 2019 Blindness Detection from Kaggle
  • Preprocessing with CLAHE and Image Augmentation
  • Transfer Learning using EfficientNet-B0
  • Achieved over 85% classification accuracy
